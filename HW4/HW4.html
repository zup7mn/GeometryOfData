Question 1

Part A

A Lie Group is a smooth manifold that also forms a group, where the two group operations, multiplication and inverse, are smooth mapping between manifolds. Let’s show that \(\text{SL}(2, \mathbb{R})\) is a Lie Group.

Showing group properites:

  1. Say \(A, B, C \in \text{SL}(2, \mathbb{R})\). Then by the rules of matrix multiplication \[ A(BC) = (AB)C \]

  2. The identity matrix \(\mathbf{I} \in \text{SL}(2, \mathbb{R})\), since \(\det \mathbf{I} = 1\).

  3. Say \(A \in \text{SL}(2, \mathbb{R})\). Since \(\det A \ne 0, A\) is invertible, and by the properties of the determinant: \[ 1 = \det A = \frac{1}{\det A^{-1}} = 1, \] so \(A^{-1} \in \text{SL}(2, \mathbb{R})\).

The \(\text{SL}(2, \mathbb{R})\) is clearly a closed subgroup of \(\text{GL}(2)\), so therefore it is a matrix group, and inherits the smooth structure of \(\text{GL}(2)\), and thus is also a Lie Group 1.

Part B

To show the homeowork’s operation is a group action, we must show that

  1. \(\mathbf{I} \cdot p = p\),
  2. \((gh) \cdot p = (g \cdot (h \cdot p))\).

The first item is trivial. To prove the second, we must recall from class that if \(z = x + i y\), then

\[ \left( \begin{pmatrix} a, b \\ c, d \end{pmatrix}, z \right) = \frac{az + b}{cz + d} = w, \] and
\[ (\Re (w), \Im (w)) = \left( \begin{pmatrix} a, b \\ c, d \end{pmatrix}, (x, y) \right). \]

This allows us to avoid some nasty algebra:

\[ \left[ \begin{pmatrix} v, w \\ x, y \end{pmatrix}, \left( \begin{pmatrix} a, b \\ c, d \end{pmatrix} , z \right) \right] = \left( \begin{pmatrix} w, x \\ y, v \end{pmatrix}, \frac{az + b}{cz + d} \right) = \frac{w(az + b) + x(cz + d)}{y(az + b) + v(cz + d)} \] \[ = \frac{(aw + cv)z + (bw + dx)}{(ay + cv) z + (by + dv)} = \left( \begin{pmatrix} aw + cz, bw + dx \\ ay + cv, by + dv \end{pmatrix}, z \right) = \left( \left[ \begin{pmatrix} v, w \\ x, y \end{pmatrix} \cdot \begin{pmatrix} a, b \\ c, d \end{pmatrix} \right], z \right) \]

So we know our operation is a group action.

First, let’s build our binary classifier and gradient descent functions. For logistic regression, our classifier takes the form \[ p_i = \frac{1}{1+e^{-(\vec{x}_i^T {\theta})}}, \] where \[ \vec{x}_i1 = 1, \] and p_i is the estimated probability that the \(i\)-th instance of our data is in class 1. We will often let \(z = \vec{x}_i^T {\theta}\). \[ p = \frac{1}{1+e^{-z}} = \frac{e^z}{e^z + 1} \iff \left( e^z + 1\right) p = e^z \] \[ \iff p e^z + p = e^z \iff p = e^z - e^ z p = e^z (1-p) \]

\[ \iff \frac{p}{1-p} = e^z. \]

We will use log likelihood to optimize our parameters: \[ \ell (\theta) = \sum y_i \log p + (1-y_i) \log \left( 1 - p \right). \]

We will rearange \(\ell (\theta)\) for the \(i\)-th prediction and do some rearangeing to help us have an easier time finding the gradient.

\[\begin{aligned} \ell_i (\theta) &= y \log p + (1-y) \log \left( 1 - p \right) \\ &= y \log p + \log \left( 1 - p \right) - y \log \left( 1 - p \right) \\ &= \log \left( 1 - p \right) + y \log \left( \frac{p}{ 1 - p} \right) \\ &= \log \left( 1 - \frac{1}{1+e^{-z}} \right) + y z \\ &= \log \left( \frac{e^{-z}}{1+e^{-z}} \right) + y z \\ &= \log e^{-z} - \log \left( 1 + e^{-z} \right) + y z \\ &= -z - \log \left( 1 + e^{-z} \right) + y z \\ &= -z - \log \left(e^{-z} \left[ e^z + 1 \right] \right) + y z \\ &= -z - \log \left(e^{-z} \right) - \log \left( e^z + 1 \right)+ y z \\ &= -z + z - \log \left( e^z + 1 \right)+ y z \\ &= y z - \log \left( e^z + 1 \right) \end{aligned}\]

Remembering that \(z = \vec{x}_i^T {\theta}\), we compute the gradient:

\[\begin{aligned} \partial_\theta \ell_i (\theta) &= \partial_\theta y_i \vec{x}_i^T {\theta} - \partial_\theta \log \left( e^{\vec{x}_i^T {\theta}} + 1 \right) \\ &= y_i \vec{x_i} - \frac{\vec{x_i} e^{\vec{x}_i^T {\theta}}}{e^{\vec{x}_i^T {\theta}} +1} \\ &= y_i \vec{x_i} - \vec{x_i} p_i \\ &= x_i (y_i - p_i). \end{aligned}\]

This is very manageable!

def sigmoid(z):
    return np.where(z >= 0, 1/(1+np.exp(-z)), np.exp(z)/(1+np.exp(z)))

class LogisticRegressor():
    def __init__(self, num_features):
        self.W = np.random.randn(num_features + 1)
    def predict(self, X: np.ndarray):
        x_b = np.insert(X, 0, 1, axis=-1)

        z = x_b @ self.W

        return sigmoid(z)
    
    def loglikelihood(self, X: np.ndarray, y: np.ndarray):
        p = self.predict(X)

        x_b = np.insert(X, 0, 1, axis=-1)

        z = x_b @ self.W

        return np.average(y * z - np.logaddexp(0, z))
    
    def train_one_step(self, X: np.ndarray, y: np.ndarray, lr = 0.01):
        p = self.predict(X)

        x_b = np.insert(X, 0, 1, axis=-1)

        z = x_b @ self.W

        G = (y - p)[:, np.newaxis] * x_b

        grad = np.mean(G, axis=0)

        likelihood = self.loglikelihood(X, y)

        self.W += lr * grad # gradient ascent

        return likelihood, G

    def train_one_epoch_natural(self, X: np.ndarray, y: np.ndarray, lr = 0.01):
        p = self.predict(X)

        x_b = np.insert(X, 0, 1, axis=-1)

        z = x_b @ self.W

        G = (y - p)[:, np.newaxis] * x_b
        grad = np.mean(G, axis=0)
        F = G.T @ G / len(y) + 1e-4 * np.eye(grad.shape[0]) # making sure our matrix stays invertible

        natural_grad = np.linalg.inv(F) @ grad
        likelihood = self.loglikelihood(X, y)
        self.W += natural_grad # * lr

        return likelihood, G

## Question 3

First, let's build our binary classifier and gradient descent functions. For logistic regression, our classifier takes the form
$$
p_i = \frac{1}{1+e^{-(\vec{x}_i^T {\theta})}},
$$
where
$$
\vec{x}_i1 = 1,
$$
and p_i is the estimated probability that the $i$-th instance of our data is in class 1. We will often let $z = \vec{x}_i^T {\theta}$.
$$
p = \frac{1}{1+e^{-z}} = \frac{e^z}{e^z + 1} \iff \left( e^z + 1\right) p = e^z 
$$
$$
\iff p e^z + p = e^z \iff p = e^z - e^ z p = e^z (1-p)
$$

$$
\iff \frac{p}{1-p} = e^z.
$$

We will use log likelihood to optimize our parameters:
$$
\ell (\theta) = \sum y_i \log p + (1-y_i) \log \left( 1 - p \right).
$$

We will rearange $\ell (\theta)$ for the $i$-th prediction and do some rearangeing to help us have an easier time finding the gradient. 

\begin{aligned}
\ell_i (\theta) &= y \log p + (1-y) \log \left( 1 - p \right) \\
&= y \log p + \log \left( 1 - p \right) - y \log \left( 1 - p \right) \\
&= \log \left( 1 - p \right) + y \log \left( \frac{p}{ 1 - p} \right) \\
&= \log \left( 1 - \frac{1}{1+e^{-z}}  \right) + y z \\
&= \log \left( \frac{e^{-z}}{1+e^{-z}} \right) + y z \\
&= \log e^{-z} - \log \left( 1 + e^{-z} \right) + y z \\
&= -z - \log \left( 1 + e^{-z} \right) + y z \\
&= -z - \log \left(e^{-z} \left[ e^z + 1 \right] \right) + y z \\
&= -z - \log \left(e^{-z} \right) - \log \left( e^z + 1 \right)+ y z \\
&= -z + z - \log \left( e^z + 1 \right)+ y z \\
&= y z - \log \left( e^z + 1 \right)
\end{aligned}

Remembering that $z = \vec{x}_i^T {\theta}$, we compute the gradient:

\begin{aligned}
\partial_\theta \ell_i (\theta) &= \partial_\theta y_i \vec{x}_i^T {\theta} - \partial_\theta \log \left( e^{\vec{x}_i^T {\theta}} + 1 \right) \\
&= y_i \vec{x_i} - \frac{\vec{x_i} e^{\vec{x}_i^T {\theta}}}{e^{\vec{x}_i^T {\theta}} +1} \\
&= y_i \vec{x_i} - \vec{x_i} p_i \\
&= x_i (y_i - p_i).
\end{aligned}

This is very manageable!


```{python}
import math
import numpy as np
# for numerical stability, we use a modified log function 
def sigmoid(z):
    return np.where(z >= 0, 1/(1+np.exp(-z)), np.exp(z)/(1+np.exp(z)))

class LogisticRegressor():
    def __init__(self, num_features):
        self.W = np.random.randn(num_features + 1)
    def predict(self, X: np.ndarray):
        x_b = np.insert(X, 0, 1, axis=-1)

        z = x_b @ self.W

        return sigmoid(z)
    
    def loglikelihood(self, X: np.ndarray, y: np.ndarray):
        p = self.predict(X)

        x_b = np.insert(X, 0, 1, axis=-1)

        z = x_b @ self.W

        return np.average(y * z - np.logaddexp(0, z))
    
    def train_one_step(self, X: np.ndarray, y: np.ndarray, lr = 0.01):
        p = self.predict(X)

        x_b = np.insert(X, 0, 1, axis=-1)

        z = x_b @ self.W

        G = (y - p)[:, np.newaxis] * x_b

        grad = np.mean(G, axis=0)

        likelihood = self.loglikelihood(X, y)

        self.W += lr * grad # gradient ascent

        return likelihood, G

    def train_one_epoch_natural(self, X: np.ndarray, y: np.ndarray, lr = 0.01):
        p = self.predict(X)

        x_b = np.insert(X, 0, 1, axis=-1)

        z = x_b @ self.W

        G = (y - p)[:, np.newaxis] * x_b
        grad = np.mean(G, axis=0)
        F = G.T @ G / len(y) + 1e-4 * np.eye(grad.shape[0]) # making sure our matrix stays invertible

        natural_grad = np.linalg.inv(F) @ grad
        likelihood = self.loglikelihood(X, y)
        self.W += natural_grad # * lr

        return likelihood, G
```
